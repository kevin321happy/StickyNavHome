# StickyNavHome
***
一个ToolBar随滑动渐变以及StickyLayout效果的首页
-----
***

1、外层布局的滑动,随着顶部图片可见高度,改变ToolBar的背景和透明度,以及背景图的透明度
 -------
2、处理外层滑动和ViewPager中Recycle列表的滑动冲突,在Sticky中根据条件拦截事件
 -------
3、指示器滑到顶部显示在ToolBar的下方,不超过ToolBar
 -------
